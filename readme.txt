Instructions:
	1. Create an input file with the following parameters. One entry per line. Capital letters only.
		Player entry: 
			PLAYER/<PLAYERID>/<FIRST NAME>/<LAST NAME>/<birthyear yyyy>/<COUNTRY>
		Match entry:
			MATCH/<player1 id>/<player2 id>/<date yyyymmdd>/<TOURNAMENT>/<scores, separated by commas e.g. 6-3,6-7,6-21>
		
	2. Input the name of the input file into the command line. Run the program. It will automatically load entries in the input file.
	3. Select desired command in the console, follow instructions on the console if input is needed.
